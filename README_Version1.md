# 🎤 Basmtak Chat - دردشة بصمتك

تطبيق دردشة صوتية متعدد المنصات مع ألعاب تفاعلية

## ✨ الميزات
- 🎤 دردشة صوتية مباشرة (Voice Chat Rooms)
- 🎮 ألعاب داخل الرومات وخارجها
- 📱 توافق كامل: iOS, Android, Linux, Windows, Mac
- 🔐 نظام تسجيل دخول آمن
- 👥 إدارة المستخدمين والرومات

## 🛠️ التكنولوجيا المستخدمة
- **Frontend**: React Native
- **Desktop**: Electron
- **Backend**: Node.js + Express
- **Realtime**: WebRTC + Socket.io
- **Database**: MongoDB / Firebase
- **Authentication**: JWT
